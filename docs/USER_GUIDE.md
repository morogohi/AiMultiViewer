# 사용자 가이드 (USER GUIDE)

AI 멀티뷰어는 여러 문서 포맷(HWPX·DOCX·PDF·TXT·Markdown)을 한 앱에서 보고,
AI로 요약·질의응답·비교·회의록 생성·기술분석까지 수행하는 안드로이드 앱입니다.

## 1. 설치

### 방법 A: GitHub Release APK (권장)
1. 저장소의 **Releases** 페이지에서 최신 `app-release.apk` 다운로드
2. 스마트폰에서 APK 열기 → "출처를 알 수 없는 앱 설치" 허용 → 설치

### 방법 B: 직접 빌드
1. Android Studio에서 프로젝트 열기 → Gradle Sync
2. USB 디버깅을 켠 스마트폰 연결 → ▶ Run

> 요구 사항: Android 8.0 (API 26) 이상

## 2. 문서 열기

1. 홈 화면 우측 하단 **[+ 문서 추가]** 버튼 탭
2. 시스템 파일 선택기에서 문서 선택 (다운로드, 드라이브 등 어디서나)
3. 홈 목록에 추가된 문서를 탭하면 뷰어가 열립니다

지원 포맷:

| 포맷 | 보기 | AI 분석 |
|------|:----:|:-------:|
| TXT | ✅ | ✅ |
| Markdown | ✅ (서식 렌더링) | ✅ |
| PDF | ✅ (페이지 렌더) | ✅ |
| DOCX | ✅ (텍스트) | ✅ |
| HWPX | ✅ (텍스트) | ✅ |
| HWP / DOC | ⏳ 후속 지원 | HWPX/DOCX로 변환 권장 |

문서 삭제: 홈 목록에서 휴지통 아이콘 탭 (원본 파일은 삭제되지 않음)

## 3. AI 기능 사용

뷰어 상단의 **[AI]** 탭에서 사용합니다.

### 요약 (설정 없이 바로 사용 가능)
**[요약]** 버튼을 누르면 온디바이스 엔진이 오프라인으로 핵심 문장을 추출합니다.
클라우드 AI를 켜면 더 자연스러운 고품질 요약을 받습니다.

### 질의응답 (Q&A)
입력창에 문서 내용에 대한 질문을 입력하고 전송하면 문서에 근거해 답변합니다.
(클라우드 AI 필요)

### 회의록 / 기술분석 / 비교
- **회의록**: 안건, 주요 논의, 결정사항, 액션아이템으로 구조화
- **기술분석**: 개요, 핵심 용어, 구성요소, 잠재 이슈 정리
- **비교**: 문서 두 개의 공통점/차이점을 표로 정리
(모두 클라우드 AI 필요)

## 4. 클라우드 AI 설정

1. 홈 화면 우측 상단 ⚙(설정) 탭
2. **클라우드 AI 사용** 스위치 켜기
3. 항목 입력:
   - **Base URL**: 기본 `https://api.openai.com/v1` (OpenAI 호환 서버면 무엇이든 가능)
   - **API Key**: 발급받은 키 (예: OpenAI `sk-...`)
   - **모델명**: 기본 `gpt-4o-mini`

> OpenAI 외에도 Azure OpenAI, 사내 LLM 서버, PC의 Ollama/LM Studio 등
> OpenAI 호환 API라면 모두 연결할 수 있습니다.
> 예) 같은 Wi-Fi의 PC에서 Ollama 실행 시: `http://<PC IP>:11434/v1`, 모델 `llama3.1`

### 개인정보 관련
- 온디바이스 요약은 문서가 기기 밖으로 나가지 않습니다.
- 클라우드 기능 사용 시 문서 본문 일부(최대 약 12,000자)가 설정한 API 서버로 전송됩니다.
- API 키는 기기에만 저장됩니다.

## 5. 자주 묻는 질문 (FAQ)

**Q. HWP 파일이 안 열려요.**
구형 바이너리 HWP는 아직 미지원입니다. 한글 프로그램에서 **HWPX로 다른 이름 저장** 후 열어주세요. DOC도 마찬가지로 DOCX 변환을 권장합니다.

**Q. 요약 품질이 아쉬워요.**
기본 요약은 오프라인 추출 방식입니다. 설정에서 클라우드 AI를 켜면 LLM 기반 고품질 요약으로 전환됩니다.

**Q. Q&A 버튼을 눌러도 "클라우드 AI가 필요합니다"라고 나와요.**
설정에서 클라우드 AI 스위치를 켜고 API 키를 입력해야 합니다.

**Q. 앱을 지웠다 다시 설치하면 문서 목록이 사라지나요?**
네, 목록은 앱 데이터에 저장되므로 삭제 시 초기화됩니다. 원본 파일은 영향받지 않습니다.

**Q. 문서를 열면 "권한이 없습니다" 오류가 나요.**
파일이 이동/삭제되었거나 URI 권한이 만료된 경우입니다. 목록에서 삭제 후 다시 추가해주세요.

## 6. 화면 구성

| 화면 | 설명 |
|------|------|
| 홈 | 추가한 문서 목록, 문서 추가/삭제, 설정 진입 |
| 뷰어 - 보기 탭 | 문서 내용 표시 (PDF는 페이지 스와이프) |
| 뷰어 - AI 탭 | 요약·회의록·기술분석·비교 버튼 + Q&A 입력창 |
| 설정 | 클라우드 AI on/off, Base URL, API Key, 모델명 |
